# Operator (연산자)

주요 연산자들의 기능과 특징을 정리합니다.

## 1. 비트 연산자 (Bitwise Operators)
비트 단위로 연산을 수행합니다.

| 연산자 | 설명 |
| :--- | :--- |
| **`&` (AND)** | 두 비트가 모두 1이면 1 반환 |
| **`\|` (OR)** | 두 비트 중 하나라도 1이면 1 반환 |
| **`^` (XOR)** | 두 비트가 서로 다르면 1 반환 |
| **`~` (NOT)** | 비트를 반전 (0→1, 1→0) |
| **`<<` (Left Shift)** | 비트를 왼쪽으로 이동시키고 오른쪽을 0으로 채움 |

---

## 2. 논리 연산자 (Logical Operators)

### 논리 AND (`&&`)
- `a && b`: `a`가 Truthy하면 `b`를 반환, Falsy하면 `a`를 반환합니다.
- 모든 값이 Truthy인 경우 마지막 값을 반환합니다.

### 논리 OR (`||`)
- `a || b`: `a`가 Truthy하면 `a`를 반환, Falsy하면 `b`를 반환합니다.

### 널 병합 연산자 (`??`)
- 왼쪽 피연산자가 `null` 또는 `undefined`일 때만 오른쪽 값을 반환합니다.
- `0`이나 `""`(빈 문자열)을 유효한 값으로 취급하고 싶을 때 유용합니다.

---

## 3. 조건 및 선택 연산자

- **삼항 연산자 (`? :`)**: `조건 ? 참일때 : 거짓일때`
- **옵셔널 체이닝 (`?.`)**: 참조가 `null` 또는 `undefined`인 경우 에러 대신 `undefined`를 반환합니다.

---

## 4. 기타 주요 연산자

- **`delete`**: 객체의 속성을 삭제합니다.
- **`instanceof`**: 객체가 특정 생성자의 인스턴스인지 확인합니다.
- **스프레드 (`...`)**: 배열이나 객체의 요소를 펼쳐서 복사하거나 전달합니다.
- **구조 분해 할당**: 객체나 배열의 값을 변수로 분해하여 할당합니다.

---

## 5. 산술 및 증감 연산자

- **거듭제곱 (`**`)**: `a ** b` (a의 b승)
- **증감 (`++`, `--`)**:
    - **전위 (`++a`)**: 값을 먼저 증가시키고 식을 평가합니다.
    - **후위 (`a++`)**: 식을 먼저 평가한 후 값을 증가시킵니다.
- **단항 더하기 (`+`)**: 숫자가 아닌 값을 숫자로 변환을 시도합니다.

---

## 6. 비교 연산자

- **일치 (`===`)**: 값과 **타입**이 모두 같은지 비교 (권장)
- **동등 (`==`)**: 타입 변환을 거친 후 값이 같은지 비교
