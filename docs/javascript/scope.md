# Scope (유효 범위)

스코프는 식별자(변수, 함수 등)에 접근할 수 있는 범위를 의미합니다.

## 1. 함수 스코프 (Function Scope)
- 함수 내에서 선언한 변수는 해당 함수 내부에서만 접근할 수 있습니다.
- `var`, `let`, `const` 모두 함수 범위를 가집니다.

## 2. 블록 스코프 (Block Scope)
- 중괄호(`{}`)로 감싸진 영역을 블록이라고 합니다.
- **`let`, `const`**: 블록 범위를 가집니다. (if, for, while, try/catch 등)
- **`var`**: 블록 범위를 가지지 않습니다. 블록 내부에서 선언해도 함수 스코프나 전역 스코프를 따르므로 의도치 않은 변수 값 변경이 발생할 수 있습니다.

```javascript
var x = 1;
{
  var x = 2; // 전역의 x가 2로 변경됨
}
console.log(x); // 2
```

---

## 3. 렉시컬 스코프 (Lexical Scope)
함수를 어디서 **호출**했는지가 아니라, 어디서 **선언**했는지에 따라 상위 스코프가 결정되는 방식입니다. 이를 **정적 스코프(Static Scope)**라고도 부릅니다.

```javascript
let a = 3;

function aa() {
  let a = 4;
  bb();
}

function bb() {
  console.log(a); // 호출 위치와 상관없이 선언될 당시의 상위 스코프를 참조
}

aa(); // 3
bb(); // 3
```

---

## 4. 스코프 체인 (Scope Chain)
현재 스코프에서 식별자를 찾지 못한 경우, 상위 스코프(외부 함수) 방향으로 계속해서 탐색하는 과정을 의미합니다. 최상위(전역) 스코프에서도 찾지 못하면 참조 에러(ReferenceError)가 발생합니다.

- **변수명 중복**: 스코프가 다르면 동일한 이름의 변수 선언이 가능하며, 현재 스코프에 가까운 식별자가 우선순위를 가집니다.
