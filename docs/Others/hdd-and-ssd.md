## SSD

플래시 메모리를 사용해 데이터를 저장하는 장치

- 플래시 메모리 : 전원이 없어도 전자를 가두거나 내보내 데이터를 저장
- 플로팅 게이트 : 전기가 흐르는 통로
    - 다른 부위와 완전히 분리된 절연체
    - 높은 전압을 가하면 전자가 절연체 장벽을 뚫고 플로팅 게이트 내부로 이동해 갇힘
    - 플로팅 게이트에 전자가 있으면 전류 흐름이 달라지는데 이 변화를 감지해 0,1로 구분
- RAM과의 차이
    - RAM은 전하가 빠르게 손실되기 때문에 데이터를 유지하려면 지속적인 전기 신호가 있어야함 따라서, 전원이 끊어지면 데이터가 사라짐
    - SSD는 플로팅 게이트에 전자를 가두거나 내보내는 방식으로 데이터를 저장하여 전원이 없어도 전자가 손실되지 않음

## HDD

회전하는 자기 저장 장치(플래터)에 데이터를 기록 (LP판과 유사)

- 데이터 쓰기 : 헤드가 플래터 표면의 자기 입자들을 N,S로 정렬시킴
- 데이터 읽기 : 헤드가 정렬된 자기 입자들을 분석
