# Typescript


타입스크립트는 자바스크립트에 타입을 부여하여 코드의 안정성을 높여주는 정적 타입 언어입니다.

## 1. 컴파일 과정
기존 컴파일러는 소스 코드를 파싱하여 **AST(Abstract Syntax Tree)**를 만들고 바이트코드로 변환하지만, 타입스크립트는 다음과 같은 과정을 거칩니다.

1. 타입스크립트 소스 코드를 파싱하여 AST 생성
2. **타입 확인(Type Check)**
3. 자바스크립트 코드로 변환(Transpile)

## 2. 특징
- **런타임 오버헤드 없음**: 타입 정보는 자바스크립트로 변환되는 과정에서 모두 제거되므로 실행 시 성능에 영향을 주지 않습니다.
- **안전한 개발 환경**: 실행 전(컴파일 단계)에 타입 에러를 발견할 수 있어 런타임 에러를 방지합니다.
